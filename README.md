# acidotic Racing Official Website

## Part of Off The Grid Coders

You can see the progress here: [LINK](https://acidotic.herokuapp.com/)

Node.js / Express / EJS
Heroku / Firebase

#### Dependencies
`gem install scss-lint`
`npm install gulp -g`

#### First Install & Run
`chmod +x install.sh && ./install.sh`

#### Running Locally
`./run.sh`

#### Backup Database (JSON)
`./backup.sh`

#### Structure
+ /bin
+ /node_modules
+ /public
  - /assets
  - /javascripts
  - /scss
  - /stylesheets
+ /routes
+ /views
  - /pages
  - /partials
